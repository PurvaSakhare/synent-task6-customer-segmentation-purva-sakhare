# Customer Segmentation using K-Means Clustering

## Project Overview

This project performs customer segmentation using the Mall Customer Dataset. K-Means Clustering is applied to group customers based on their annual income and spending behavior. The analysis helps businesses understand different customer groups and create targeted marketing strategies.

## Objectives

* Perform data preprocessing
* Apply K-Means Clustering
* Identify customer segments
* Visualize customer clusters
* Generate business insights

## Dataset

The dataset contains the following attributes:

* CustomerID
* Gender
* Age
* Annual Income (k$)
* Spending Score (1-100)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Google Colab

## Project Workflow

### 1. Data Preprocessing

* Loaded the dataset
* Checked for missing values
* Selected relevant features for clustering

### 2. K-Means Clustering

* Used the Elbow Method to determine the optimal number of clusters
* Applied K-Means algorithm to segment customers

### 3. Data Visualization

* Visualized customer clusters using scatter plots
* Displayed cluster centroids for better interpretation

## Customer Segments

### Cluster 1

Low Income – Low Spending Customers

### Cluster 2

High Income – High Spending Customers

### Cluster 3

High Income – Low Spending Customers

### Cluster 4

Low Income – High Spending Customers

### Cluster 5

Average Income – Average Spending Customers

## Key Insights

* Premium customers belong to the High Income – High Spending segment.
* High Income – Low Spending customers can be targeted through personalized marketing campaigns.
* Low Income – High Spending customers respond well to promotional offers.
* Customer segmentation improves marketing effectiveness and customer retention.
